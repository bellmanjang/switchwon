# [스위치원] 프론트엔드 개발자 사전 과제: 환전 애플리케이션

---

## 사용 도구 및 라이브러리

- **ChatGPT**

  프로젝트 구조 설계, 컴포넌트 분리 전략, 코드 리뷰 및 개선 아이디어 도출에 활용

- **react-router-dom**

  SPA 기반 클라이언트 사이드 라우팅

  로그인/환전/거래내역 페이지 분리 및 인증 상태에 따른 보호 라우트(Guard) 구현

- **@radix-ui/themes**

  디자인 시스템 & UI 컴포넌트

- **tailwindcss, @tailwindcss/vite**

  유틸리티 기반 CSS 프레임워크 및 Vite 플러그인

- **lucide-react**

  SVG 아이콘 컴포넌트

- **prettier**

  일관된 코드 스타일 유지를 위한 코드 포맷터

- **vite**

  빠른 개발 환경을 위한 빌드 도구 및 번들러

- **vite-tsconfig-paths**

  tsconfig의 path alias 설정을 Vite에 자동으로 적용하기 위한 플러그인
